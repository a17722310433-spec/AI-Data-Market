## 👤 用户端功能

### 2. 用户注册

**访问路径**: `/register`

**功能说明**:
- 支持个人用户和企业用户两种角色
- 表单验证（邮箱格式、密码强度）
- 第三方登录（Google、GitHub）
- 服务条款同意

**操作步骤**:
1. 选择注册类型（个人/企业）
2. 填写用户名/公司名称
3. 填写邮箱和密码
4. 同意服务条款
5. 点击"创建账户"

**演示要点**:
- 表单验证动画
- 角色切换效果

---

### 3. 用户登录

### 4. 用户仪表盘

**访问路径**: `/dashboard`

### 5. 上传数据

**访问路径**: `/dashboard/upload`

**功能说明**:
- 支持多文件上传
- 自动识别AI平台
- 拖拽上传
- 实时上传进度
- AI自动评估

**支持格式**:
- `.json` - ChatGPT/Claude导出
- `.html` - ChatGPT导出
- `.txt` / `.md` - 通用格式
- `.csv` - 表格数据

**操作步骤**:
1. 选择AI平台（ChatGPT/Claude/文心一言）
2. 拖拽或选择文件
3. 点击"上传并评估"
4. 等待AI评估完成
5. 查看评估结果

**评估结果展示**:
- 数据质量分数
- 预估价值
- 识别的分类
- 标签列表

**演示要点**:
1. 拖拽上传动画
2. 上传进度条
3. 评估结果卡片

## 💳 支付流程

### 14. Stripe集成

**购买流程**:
1. 用户选择数据 → 点击购买
2. 创建Stripe Checkout Session
3. 跳转到Stripe支付页面
4. 完成支付
5. Webhook通知更新订单
6. 数据解锁访问

**提现流程**:
1. 用户申请提现
2. 后端验证余额
3. 创建Stripe Payout
4. 资金转入银行账户

---

## 🤖 AI评估系统

### 15. 数据评估

**评估维度**:
1. **消息数量** - 对话长度
2. **内容复杂度** - 语义丰富度
3. **主题分类** - 自动识别领域
4. **标签生成** - 关键词提取
5. **质量评分** - 综合打分
6. **价值估算** - 价格建议

**评估流程**:
```
上传文件 → 解析内容 → 发送给OpenAI API → 分析结果 → 生成报告
```
