## 第二步：注册 MongoDB Atlas 免费版

### 2.1 访问 MongoDB Atlas

1. 打开手机浏览器
2. 访问：**cloud.mongodb.com**
3. 点击 **Get started free**

### 2.2 创建账号

```
1. 点击 "Start free"
2. 输入邮箱地址
3. 设置密码（至少8位，包含大小写和数字）
4. 同意服务条款
5. 点击 "Create Free Account"
```

### 2.3 验证邮箱

```
1. 去邮箱查收验证邮件
2. 点击邮件中的验证链接
3. 等待自动跳转回 Atlas 控制台
```

### 2.4 创建免费集群（关键步骤）

#### 选择免费套餐
```
1. 选择 "Shared Clusters" (免费)
2. 选择服务器区域：
   - 推荐亚洲：Singapore (新加坡)
   - 或 Mumbai (孟买)
   ⚠️ 不要选择太远的区域，会影响速度
3. 点击 "Create Cluster"
```

#### 等待集群创建
```
⏱️ 需要等待 1-3 分钟创建完成
📱 界面会显示进度条
```

### 2.5 配置数据库访问权限

#### 创建数据库用户

```
1. 在左侧菜单点击 "Database Access"
2. 点击 "Add New Database User"
3. 填写信息：
   - Authentication Method: Password
   - Username: your_app_user
   - Password: [生成一个强密码，保存备用]
   - Database Privileges: Read and write to any database
4. 点击 "Add User"
```

#### 配置网络白名单

```
1. 在左侧菜单点击 "Network Access"
2. 点击 "Add IP Address"
3. 点击 "Allow Access from Anywhere"
4. 点击 "Confirm"
```

⚠️ **重要**：这一步允许 Vercel 服务器连接数据库

### 2.6 获取连接字符串

```
1. 返回 "Clusters" 页面
2. 点击 "Connect" 按钮
3. 选择 "Connect your application"
4. 复制 Connection String
5. 格式类似：
   mongodb+srv://your_app_user:<password>@cluster0.xxxxx.mongodb.net
```

⚠️ **注意**：将 `<password>` 替换为刚才设置的密码

## 第五步：一键部署到 Vercel

### 5.1 从 GitHub 部署

```
1. 在 Vercel 控制台点击 "Add New Project"
2. 选择 "Import Git Repository"
3. 找到并选择刚 Fork 的项目
4. 点击 "Import"
```

### 5.2 配置环境变量（关键步骤）

在 "Environment Variables" 部分添加以下变量：

#### 必需的环境变量

```
┌──────────────────────────────────────────────────────────────┐
│  VARIABLE_NAME                    VALUE                      │
├──────────────────────────────────────────────────────────────┤
│  MONGODB_URI                       mongodb+srv://user:pass@.. │
│  STRIPE_PUBLISHABLE_KEY           pk_test_xxxxx              │
│  STRIPE_SECRET_KEY                 sk_test_xxxxx              │
│  STRIPE_WEBHOOK_SECRET             whsec_xxxxx                │
│  NEXT_PUBLIC_SITE_URL              https://your-app.vercel.app│
└──────────────────────────────────────────────────────────────┘
```

#### 各变量获取方式

| 变量名 | 获取方式 |
|--------|---------|
| `MONGODB_URI` | MongoDB Atlas 连接字符串 |
| `STRIPE_PUBLISHABLE_KEY` | Stripe Dashboard → Developers → API keys |
| `STRIPE_SECRET_KEY` | Stripe Dashboard → Developers → API keys |
| `STRIPE_WEBHOOK_SECRET` | Stripe Dashboard → Developers → Webhooks |
| `NEXT_PUBLIC_SITE_URL` | Vercel 部署后会生成，先填 `https://placeholder.vercel.app` |

### 5.3 完成部署

```
1. 点击 "Deploy"

3. 获得访问域名：your-app.vercel.app
```

### 5.4 更新站点URL

部署成功后：

## ❓ 常见问题

### Q1: 手机浏览器打不开 Vercel/MongoDB？
```
解决方案：
1. 确保网络稳定
2. 尝试切换 4G/WiFi
3. 清除浏览器缓存
4. 尝试使用 Chrome 浏览器
```

### Q2: MongoDB 连接失败？
```
检查清单：
1. ✓ 网络白名单是否设置为 0.0.0.0/0
2. ✓ 用户名密码是否正确
3. ✓ 密码中没有特殊字符或已正确URL编码
4. ✓ 集群状态是否为 "Available"
```

### Q3: Stripe 支付不成功？
```
检查清单：
1. ✓ 使用的是测试卡号 4242 4242 4242 4242
2. ✓ API Keys 是 pk_test_ 和 sk_test_ 开头
3. ✓ 未使用真实信用卡
```

### Q4: 部署后网站显示空白？
```
解决方案：
1. 检查环境变量是否全部填写
2. 查看 Vercel Build 日志排查错误
3. 尝试重新部署
```

### Q5: 如何绑定自定义域名？
```
1. 在 Vercel 项目 Settings → Domains
2. 添加你的域名
3. 按提示配置 DNS
4. 等待验证通过
```
